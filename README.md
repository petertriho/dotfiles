# dotfiles

## Installation

### Dependencies

#### GNU Stow

- MacOS (homebrew): `brew install stow`
- Arch (pacman): `sudo pacman -S stow`
- Ubuntu (apt): `sudo apt install stow`

### Instructions

1. `git clone https://github.com/petertriho/dotfiles.git ~/.dotfiles`
2. `cd ~/.dotfiles; stow --no-folding -t ~ */`
